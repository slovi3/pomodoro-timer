# Pomodoro Timer Pro (Vanilla JS)

Şık arayüze sahip bir pomodoro timer uygulaması. Focus ve break modları arasında geçiş yapar, session sayaçlarını localStorage ile saklar.

## Özellikler
- Focus 25 / Break 5 / Break 15 presetleri
- Start / Pause / Reset
- Completed sessions sayacı
- Today sayacı
- localStorage kalıcılığı
- Responsive modern UI

## Çalıştırma
`index.html` dosyasını çift tıklayarak tarayıcıda aç.
